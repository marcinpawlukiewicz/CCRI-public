## Program 2: Factorials [4 PTS]

> **DEFINITION:**
> A factorial, denoted by \(n!\), is the product of all positive integers less than or equal to a given non-negative integer \(n\). Defined for non-negative integers, \(n!=n _ (n-1) _ (n-2) _ ... _ 1\). For example, 5!=120=5 _ 4 _ 3 _ 2 _ 1, and \(0!\) is defined as \(1\).

Create a program that computes any factorial given by the user that meet's given requirements.

### Requirements:

- If factorial's value reaches `2147483647` or is equal or smaller than `0` the output should be: `Can't handle this`.
- If input's user is `0`, expected output is `1`.
- Any other input should be computed according with the definition of factorials.
