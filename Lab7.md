## Overview

This lab is more self-taught and is meant to give you a better understanding of HTML, CSS, and how they relate to JavaScript. The best way to learn both is through hands-on practice. HTML and CSS can be tricky because what you write is not always exactly what you see in the browser.

> **Note:** Please use Google Chrome for this lab. Different browsers may display the same page a little differently. More advanced developers account for those differences with additional CSS, but for this lab we want to keep the environment simple and consistent for grading.

## Part 2A - Website (HTML)

Create `index.html`, `style.css`, and an `img/` directory. Build a website of your choosing that contains the following:

- Start with the minimal HTML skeleton from slide 7: `<!doctype html>`, `<html lang="en">`, `<head>`, and `<body>`.
- Include a header section for the page.
- Add an icon image and place it in your main project directory or another location that you reference correctly.
- Use at least two different heading tags.
- Use at least three `<p>` elements.
- Use at least three of the following tags: `<br>`, `<hr>`, `<strong>`, `<em>`, `<span>`.
- Add two links to `google.com`: one should open in a new tab, and one should open in the same tab.
- Use an `<img>` tag with an image of your choosing. The image should be downloaded and stored in `img/`.
- Create one list with 5 items.
- Add a `<footer>` and include your name in it. The footer should be at the bottom of the page.
- Use at least one `<button>` and at least one other form tag from slide 12, such as `<label>`, `<input>`, `<textarea>`, `<select>`, or `<option>`.
- Create a 5x7 table and fill it in.
